# 张家琛工作任务总结
问题：本地gitbook分支push 上去与主城的分支合并有冲突
原因：本地的远端通信地址链接的是自己github上gitbook库，所以每次pull都是自己的
解决思路：删除gitbook本地的通信地址，添加主城的gitbook库为通信地址。
然后在master分支git checkout -b 新分支， git checkout 新分支，修改好后，就可以push了。
